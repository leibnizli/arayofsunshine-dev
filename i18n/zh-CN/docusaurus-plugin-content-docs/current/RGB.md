# RGB

一款取色App，支持macOS，可在屏幕任意位置取色。

![RGB-App-Store](https://user-images.githubusercontent.com/1193966/72804897-90c2e080-3c8c-11ea-88e7-b124a1be9fe7.png)

选择颜色后，十六进制格式代码将自动复制到剪贴板。
