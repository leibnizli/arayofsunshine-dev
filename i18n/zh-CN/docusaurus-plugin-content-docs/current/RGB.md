# RGB

一个取色App，支持macOS，可在屏幕任意位置取色。

![RGB-App-Store](https://user-images.githubusercontent.com/1193966/169992576-3460712c-3800-4d5a-b636-ab9371835c6a.png)

选择颜色后，十六进制格式代码将自动复制到剪贴板。
