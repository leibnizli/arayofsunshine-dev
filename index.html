Helm

Helm is a hosts file manager that runs on macOS. Helm enables you to edit hosts files and switch between them. Helm merges the original file and the active file into one file each time you switch. Write to the hosts file of the mac system.

FontMagic

FontMagic is a font manipulation application that supports macOS. It can export specific glyphs to SVG, PDF and PNG files. The color and size of the output can be set in the preferences.

RGB

A color picking app that supports macOS and can pick colors anywhere on the screen.

PCalculator

A simple calculator that supports addition, subtraction, multiplication and division.
